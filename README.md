# bamboo-server

GSM 대나무숲 서버

보안이슈의 경우 `qudwls185@naver.com`으로 부탁드려요!  
많은 이슈와 풀리퀘스트, 그리고 피드백 부탁드려요!

## 시작하기

- install package

```sh
$ yarn
```

- write .env (refer .env.example)

```env
PORT=
MONGO_URL=
DISCORD_WEBHOOK=
DISCORD_TEST_WEBHOOK=
DISCORD_MANAGEMENT_WEBHOOK=
ADMIN_PASSWORD=
JWT_SECRET=
```

- start

```sh
$ yarn local
```
