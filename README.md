## 시작하기

- install package

```sh
$ yarn
```

- write .env (refer .env.example)

```env
MONGO_URL=
DISCORD_WEBHOOK=
DISCORD_TEST_WEBHOOK=
DISCORD_MANAGEMENT_WEBHOOK=
ADMIN_PASSWORD=
JWT_SECRET=
```

- start

```sh
$ yarn local
```
